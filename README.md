# 월급 지급명세서 관리 템플릿

요청하신 표 형태를 기준으로, 매월 직원 지급명세서를 쉽게 관리할 수 있는 단일 HTML 템플릿입니다.

## 사용 방법

1. `index.html`을 브라우저에서 열기
2. 대상 월 선택
3. 직원 행 수정/추가
4. `저장` 버튼으로 브라우저 로컬 저장
5. `CSV 내보내기` 버튼으로 파일 다운로드

## 포함 기능

- 이미지 양식과 유사한 큰 표 헤더/그리드 스타일
- 기본 샘플 데이터(예시 인원)
- 행 추가/삭제
- 주휴수당(시급 × 주휴시간) 자동 계산
- 지급총액(급여 + 주휴수당) 자동 계산
- 총 지급액 자동 합산
- 월별 저장(로컬스토리지)
- CSV 내보내기(엑셀 호환용 UTF-8 BOM)
